Vault

 

The Vault window is used to display a list of items that are located in a personal or house vault.

 

This window appears when you right click on a vault NPC.  It will also appear when you right click on a vault in housing that you have access to.  You can add and remove an item from a vault by performing drag and drop operations between the vault and your characters backpack.  The vault window is resizeable.  The Toggle Page Mode button in the window is used to switch between list and page modes.  In list mode you can resize the window or use the scroll bar to see all available items in the vault.  In page mode you can use the left and right arrow buttons to switch between vault pages to see all the items.

 

- Conceal

There is nothing shown when the window is concealed.  The window is invisible with no graphics or controls to minimize the amount of memory used.  Nothing will appear on the screen when you attempt to activate the window.
