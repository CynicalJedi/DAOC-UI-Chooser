Menu Bar

 

The Menu Bar window is used to display status information and provide some command buttons to access various windows.

 

The window will appear when you click on the button labeled Menu Bar from the Interface Options window of the User Interface.

 

- Conceal

There is nothing shown when the window is concealed.  The window is invisible with no graphics or controls to minimize the amount of memory used.  Nothing will appear on the screen when you attempt to activate the window.
