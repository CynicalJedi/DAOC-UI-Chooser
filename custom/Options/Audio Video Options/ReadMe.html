Audio and Video Options

 

The Audio and Video Options window is used to display the current volume, graphics, camera, name, and quickbar settings.

 

The window will appear when you click on the button labeled "Options" in the Main Menu or Command windows of the User Interface.

 

- Conceal

This option caused the game client to crash so it was disabled.

 

- Disable

This option caused the game client to crash so it was disabled.
