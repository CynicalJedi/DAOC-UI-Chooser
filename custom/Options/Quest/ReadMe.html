Quest

 

The Quest Start and Finish windows are used to interact with new quests.

 

The windows will appear automatically when you right click on a quest NPC to start or finish a new quest.  The windows display text about the quest along with reward information.  You can select the item icon for an award and delve it to get additional information about the items being offered.  The Accept button is used to accept a quest.  The Finish button is used to finish the quest and accept the selected reward.  The Cancel and Close buttons are used to close the quest window.

 

- Conceal

There is nothing shown when the windows are concealed.  The windows are invisible with no graphics or controls to minimize the amount of memory used.  Nothing will appear on the screen when you attempt to activate the windows.
