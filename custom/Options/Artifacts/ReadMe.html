Artifacts

 

The Artifacts window is a tabbed custom window used to show information about the various artifacts that can be obtained from the Trials of Atlantis areas.  Each artifact is listed in the window along with the general task and location of the artifact encounter.  The location of where to farm for the each of the scrolls needed to activate the artifacts is also provided.  A listing of the completed scroll book names and the bounty point purchase prices for the artifacts and books is also provided.

 

The window will appear when you click on the button labeled "Artifacts" in the Main Menu or Command windows of the user interface.

 

- Conceal

There is nothing shown when the window is concealed.  The window is invisible with no graphics or controls to minimize the amount of memory used.  Nothing will appear on the screen when you attempt to activate the window.
