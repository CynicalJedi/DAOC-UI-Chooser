Note:  The Artifacts window is rather large and takes up a lot of screen space.  Because of this it is not recommended that you enable the window unless you have your screen resolution set to at least 1024x768.  The artifact tables used in this window are the same as those used in other UIs such as the EggiSohwyn and Emerikol UIs.
