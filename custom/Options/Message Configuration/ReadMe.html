Message Configuration

 

The Message Configuration window is used to change the type and color of message that appear in the selected chat window.

 

The window will appear when you right click on a chat window and select the Messages button.  Click on the button beside the message types you want shown in the current selected chat window.  Select a message type and click on the color at the bottom of the window to change the color of the message.  The Accept button is used to save any changes and close the window.  The Cancel button is used to ignore any changes and close the window.

 

- Conceal

There is nothing shown when the window is concealed.  The window is invisible with no graphics or controls to minimize the amount of memory used.  Nothing will appear on the screen when you attempt to activate the window.
