Healer Helper

 

The Healer Helper windows are used to show health, endurance, and power levels for your group mates.  The windows can serve as an aid by various classes to maintain the levels at their maximum values.  These options allow you to independently change the Exterior Overlay, Interior Overlay, and Style for the Healer Helper windows.  In addition you can choose "widgets" that will appear in the group member area of the windows when a player is mezzed, diseased, poisoned, etc.

 

The windows will appear when you click on the buttons labeled Cube, Horizontal, and Vertical in the Main Menu or Command windows of the user interface.

 

Exterior Overlay

These options allow you to change the style of the exterior overlay used in the Healer Helper windows.

 

Interior Overlay

These options allow you to change the style of the interior overlay used in the Healer Helper windows.

 

Style

These options allow you to change the style of the Healer Helper windows.

 

Widgets

These options allow you to change the style of the widgets used in the Healer Helper windows.
