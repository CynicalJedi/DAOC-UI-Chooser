Healer Helper - Style - Vertical

 

These options are used to change the style of the Vertical Healer Helper window.

 

Note: The mini style windows (style 3 and 4) appear best when used with the Gem, Globe, Heart, Left Half, Collapsing Squares, Man, or Pie Chart Interior Overlays.

 

- Conceal

There is nothing shown when the window is concealed.  The window is invisible with no graphics or controls to minimize the amount of memory used.  Nothing will appear on the screen when you attempt to activate the window.
