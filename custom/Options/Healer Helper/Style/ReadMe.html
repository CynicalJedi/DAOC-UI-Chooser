Healer Helper - Style

 

These options allow you to change the style of the Healer Helper windows.

 

Basic

These options are used to change the "Basic" style of the Healer Helper window.  The style chosen will determine the basic appearance for the Cube, Horizontal, and Vertical healer helper windows.

 

Cube

These options are used to change the style of the Cube Healer Helper window.

 

Horizontal

These options are used to change the style of the Horizontal Healer Helper window.

 

Vertical

These options are used to change the style of the Vertical Healer Helper window.
