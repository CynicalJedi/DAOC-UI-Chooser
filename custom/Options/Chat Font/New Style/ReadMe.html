Chat Font - New Style (True Type)

 

These are the New Style Chat Fonts.

 

What is unique about these fonts is the game allows you to adjust the size of them as you play using the Font Resize menu option from the popup that appears when you right click in the chat window.  The screen shots for each of the images below were taken with the font size set at 10 points.  In some cases the fonts may appear "fuzzy" at this point size.  It may be necessary to increase the point size or choose a different dialog skin for better appearance of the font in the game

 

Note 1:  Some players have complained about lag that occurs when the chat window updates when using one of the new style fonts.  This is caused by the method the game client uses to draw the font images in the chat window.  If you experience this problem, you can select one of the Old Style vs New Style chat fonts

 

Note 2:  Unlike other UI changes, the game client will not reload the new style fonts used with the Chat window when you rezone. You must logout and quit the game first, then restart the game and log back in to see New Style chat font changes.
