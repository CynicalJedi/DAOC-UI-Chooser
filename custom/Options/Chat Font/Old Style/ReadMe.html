Chat Font - Old Style (Bitmap)

 

These are the Old Style Chat Fonts.

 

Unlike the New Style fonts, you cannot adjust the size of Old Style fonts as you play.  However, you can select from various font sizes using this program before you login or rezone.  In some cases the fonts may appear "fuzzy" while in the game.  It may be necessary to increase the point size, choose a different color for the chat message types, or choose a different dialog skin for better appearance of a particular font in the game.

 

Note:  The font used in the text memo areas of the Help, Information, Interact, Map, Master Level, New Quest Start, and New Quest Finish windows will change to match the style of the last "old style" chat font selected.
