Armor Resists

 

The Armor Resist window is a custom window used to show information about the armor types worn by the various classes in the game.  It indicates the highest type of armor each class can wear.  In addition it indicates the Melee, Magic (bolt), and Elemental (legendary) damage types the armor is neutral, resistant, and vulnerable against.  The information can be used to determine how to maximize your damage against a particular class of characters in the game.

 

The window will appear when you click on the button labeled "Armor Res" in the Main Menu or Command windows of the user interface.

 

- Conceal

There is nothing shown when the window is concealed.  The window is invisible with no graphics or controls to minimize the amount of memory used.  Nothing will appear on the screen when you attempt to activate the window.

 

- Disable
