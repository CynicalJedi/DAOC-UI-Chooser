Summary - Style - Normal

 

These are the Normal style Summary windows.

 

Style 1

This style shows the player health, endurance, and power status using colored bars inside the window.

 

 

Style 2

This style shows the player health, endurance, and power status using colored bars inside the window.  In addition the target name and health status are shown at the bottom of the window.

 

 

Style 3

This style shows the player health, endurance, power and concentration status using colored bars inside the window.

 

 

Style 4

This style shows the player health, endurance, power and concentration status using colored bars inside the window.  In addition the target name and health status are shown at the bottom of the window.

 

 

Style 5

This style shows the player health, endurance, Power, and XP status using colored bars inside the window.  You can change the type of XP displayed by clicking on the XP status bar.  The XP type will toggle between Exp, Rxp, Mxp, and Cxp.

 

 

Style 6

This style shows the player health, endurance, Power, and XP status using colored bars inside the window.  You can change the type of XP displayed by clicking on the XP status bar.  The XP type will toggle between Exp, Rxp, Mxp, and Cxp.  In addition the target name and health status are shown at the bottom of the window.

 

 

Style 7

This style shows the player health, endurance, power and XP status using colored bars inside the window.  In addition the player Realm Rank information is shown at the bottom of the window.

 

 

Style 8

This style shows the player health, endurance, power, concentration, and XP status using colored bars inside the window.  In addition the player Realm Rank information is shown at the bottom of the window.

 

 

Style 9

This style shows the player health, endurance, power, and XP status using colored bars inside the window.  In addition the player Realm Rank, Master Level, and Champion Level information is shown at the bottom of the window.

 

 

Style 10

This style shows the player health, endurance, power, concentration, and XP status using colored bars inside the window. In addition the player Realm Rank, Master Level, and Champion Level information is shown at the bottom of the window.

 

 

 

Notes:  The status bar skins can be changed using the choices in the Status Bar Skins option.  There are some invisible buttons in the normal Summary windows.  Click on the various status bars to activate them.  Click on the images to either side of the player name to toggle the floating play health, endurance, power, and target health windows on/off.
