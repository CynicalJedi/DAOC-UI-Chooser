Summary - Style - Stained

 

These are the Stained style Summary windows.

 

Style 1

This style shows the player concentration, health, power and endurance status using colored bars inside the window.

 

 

Style 2

This style shows the player concentration, health, power and endurance status using colored bars inside the window.

 

This style also has a built in compass and performance meter. The meter shows performance (packet loss) status on the left, lag (ping time) status in the middle, and frame rate (frames per second) status on the right. The indicators change from green to yellow and then red as the lag increases and the performance/frame rate drops.
