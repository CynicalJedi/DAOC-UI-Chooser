Summary - Style - Moewen

 

These are the Moewen style Summary windows.

 

Style 1

This style shows the player endurance, health, and power status using colored bars inside the window.

 

 

Style 2

This style shows the player endurance, health, and power status using colored bars inside the window. In addition the target name and health status are shown at the bottom of the window.

 

 

Style 3

This style shows the player PVE XP, endurance, health, power, and concentration status using colored bars inside the window.

 

 

Style 4

This style shows the player PVE XP, endurance, health, power, and concentration status using colored bars inside the window. In addition the target name and health status are shown at the bottom of the window.

 

 

Style 5

This style shows the player master level XP, endurance, health, power, and champion level XP using colored bars inside the window.

 

 

Style 6

This style shows the player master level XP, endurance, health, power, and champion level XP using colored bars inside the window. In addition the target name and health status are shown at the bottom of the window.

 

 

Style 7

This style shows the player master level XP, endurance, health, power, and champion level XP using colored bars inside the window. In addition the player PVP information is shown at the bottom of the window.

 

 

Style 8

This style shows the player concentration, endurance, health, power, and champion level XP using colored bars inside the window. In addition the player PVP information is shown at the bottom of the window.
