How To

 

The Option tree list to the left shows all options available with this user interface.

 

How to Select an Option

Click on an option name to select it.  The text in this area will change to show information about the current selected option.

 

 

How to Expand/Collapse an Option

Double click on an option name to expand it.  Double click on the option name again to collapse it.

 

 

How to Expand/Collapse ALL Options

Click on the plus and minus buttons at the top of the option list to expand or collapse all levels of the list.

 

 

How to Change an Option

Click on the bubble checkmark next to an option to change the selected option.  A small popup window indicating changes are being processed will appear.  The options currently selected will appear with a green bubble checkmark beside them.  The options designated as defaults but not selected will appear with a dark gray checkmark beside them.  Other options not selected will appear with a light gray checkmark beside them.
