Status Index

 

The Status Index window has several buttons which are used to access the Status Attributes, Inventory, Special Abilities, Combat Styles, Spells, Group, and Main Menu windows.

 

The window is static and will always appear on your display unless it has been disabled.

 

Background

There are two types of backgrounds for the Status Index window that you can change.  These options allow you to choose the background style.

 

Style

There are several styles for the Status Index window that you can choose.  These options allow you to choose the style for the Status Index window.
