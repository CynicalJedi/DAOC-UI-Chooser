Stats Group

 

The Group window is used to display the status of group members.

 

The window will appear when you click on the Group button on the Status Index or Menu Bar windows of the User Interface.

 

Note: This window is resizeable.  You can stretch the window vertically to see additional items in the list.

 

- Conceal

There is nothing shown when the window is concealed.  The window is invisible with no graphics or controls to minimize the amount of memory used.  Nothing will appear on the screen when you attempt to activate the window.
