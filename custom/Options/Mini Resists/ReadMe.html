Mini Resists

 

The Mini Resists is a custom window used to display information about a players resists.

 

The window will appear when you click on the button labeled "Mini Resists" in the Main Menu or Command windows of the user interface. It will also appear if you click on the resists area of the Status Attributes window.

 

- Conceal

There is nothing shown when the window is concealed.  The window is invisible with no graphics or controls to minimize the amount of memory used.  Nothing will appear on the screen when you attempt to activate the window.
