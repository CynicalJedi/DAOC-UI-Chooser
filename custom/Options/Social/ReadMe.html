Social

 

The Social window is a tabbed window used to view the latest News, display Guild information and see status of guild members, display Alliance information and see status of alliance members, display your friends currently online, and to display a list of player currently looking for a guild.

 

The window will appear when you click on the button labeled "Social" in the Main Menu or Command windows of the user interface.

 

- Conceal

There is nothing shown when the window is concealed.  The window is invisible with no graphics or controls to minimize the amount of memory used.  Nothing will appear on the screen when you attempt to activate the window.
