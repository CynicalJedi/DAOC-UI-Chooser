Clock

 

The Clock window is used to display the current real time.  In addition to the real time, it can be used to display the game server time, a compass, performance meter, or some combination of them.

 

The window will appear when you click on the button labeled "Clock" in the Main Menu or Command windows of the user interface.

 

- Conceal

There is nothing shown when the window is concealed.  The window is invisible with no graphics or controls to minimize the amount of memory used.  Nothing will appear on the screen when you attempt to activate the window.

 

- Disable

 

 

Style 1

This style Clock window displays the current real time.

 

 

Style 2

This style Clock window displays the current real time with a compass.

 

 

Style 3

This style Clock window displays the current real time along with a performance meter.  The performance meter is arranged in normal order representing Performance (Packet Loss), Lag (Ping Time), and then Frame (Frames per Second).  The indicators change from green to yellow then red as the lag increases and performance/frame rate drops.

 

 

Style 4

This style Clock window displays the current real time and game server time.

 

 

Style 5

This style Clock window displays the current real time, game server time, compass, and performance meter

 

 

Style 6

 

This style Clock window displays the current game server time and performance meter.
