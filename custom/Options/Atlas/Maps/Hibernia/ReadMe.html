Atlas - Maps - Hibernia

 

These options allow you to disable or set the style of the Hibernia realm Classic, SI, Catacombs, DR-Epic, and Housing atlas maps.

 

When an Atlas map set is enabled the names of each map in that set will be displayed in the window.  When an Atlas map set is disabled only the realm symbol for that set will be displayed.

 

- Disabled

When the map set is disabled, only the realm symbol will be shown.
