Mini Pet

 

The Mini Pet is a custom window used to display information and control a player's pet.

 

The window will appear when you click on the button labeled "Mini Pet" in the Main Menu or Command windows of the user interface.  It will also appear automatically when a pet is summoned.

 

- Conceal

There is nothing shown when the window is concealed.  The window is invisible with no graphics or controls to minimize the amount of memory used.  Nothing will appear on the screen when you attempt to activate the window.
