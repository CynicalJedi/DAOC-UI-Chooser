-[Icon Pack 2]-

 

There are five icon sets that you can independently change.

 

Note:  This is a transparent version of the original Icon Pack supplied with BobsUI.  Many of the icon sets may look similar in the screenshots because the background was removed.

 

Combat

This set of icons is used to display the various "Melee Styles" that a character can perform.

 

 

Items

This set of icons is used to display the various "Items" that can be found in the game.

 

 

Skills

This set of icons is used to display the various "Skills" that a character can have.

 

 

Spells

This set of icons is used to display the various "Spells" that a character can cast.

 

 

Weapons

This set of icons is used to display the various "Weapons" that a character can wield.

 

 

 

Note: The above images are only sample styles and other variations are available to choose.
