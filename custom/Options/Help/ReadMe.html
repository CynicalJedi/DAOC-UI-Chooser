Help

 

The Help window is used to display interactive help information.

 

The window will appear when you click on the button labeled "Help" in the Main Menu or Command windows of the user interface.  In addition you can type the command "/help" to activate the window.  To navigate the help system click on the highlighted text in brackets.  You can toggle the Show Login radio button to indicate if you want the Help window displayed each time you login.  The Close button is used to close the window.

 

- Conceal

There is nothing shown when the window is concealed.  The window is invisible with no graphics or controls to minimize the amount of memory used.  Nothing will appear on the screen when you attempt to activate the window.
