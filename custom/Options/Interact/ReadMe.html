Interact

 

The Interact window is used to display interactive information.

 

The window will appear automatically whenever interactive information needs to be displayed such as when clicking on a Channler NPC.

 

- Conceal

This option caused the game client to crash when an interactive event occurs so it was disabled.

 

- Disable

This option caused the game client to crash when an interactive event occurs so it was disabled.
