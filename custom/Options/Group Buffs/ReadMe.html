Group Buffs

 

The Group Buffs window is used to display each of your group members and their associated buffs.

 

The window will appear when you click on the button labeled "Grp Buffs" in the Main Menu or Command windows of the user interface.

 

- Conceal

There is nothing shown when the window is concealed.  The window is invisible with no graphics or controls to minimize the amount of memory used.  Nothing will appear on the screen when you attempt to activate the window.

 

- Disable

 

 

Style 1

 

 

Style 2

This style window includes group members health, endurance, and power status.

 

 

Style 3

This style window includes group members health, endurance, and power status.  In addition, the status of your characters concentration buffs and concentration level is shown.

 

 

Style 4

This style window is re-sizable so you can adjust it depending on the number of group members.

 

 

Style 5

This style window is tabbed and shows the buffs for only one group member at a time.
