Stats Spells

 

The Spells window is used to display your character's magic spells.  Drag and drop spells from the list to your quickbar in order to use them.

 

The window will appear when you click on the Spells button on the Status Index or menu Bar windows of the User Interface.

 

Note:  This window is resizeable.  You can stretch the window vertically to see additional items in the list.
