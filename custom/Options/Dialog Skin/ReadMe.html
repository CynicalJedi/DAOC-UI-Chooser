Dialog Skin

 

Dialogs are the actual windows that are used to display and gather information.  These options allow you to alter the frames and skin or texture used with the dialog windows.

 

Frame

There are two types of frames for the Dialog windows that you can independently change.  These options allow you to choose the frame styles.

 

Style

There are several styles for the Dialog skins that you can choose.  These options allow you to choose the skin style.
