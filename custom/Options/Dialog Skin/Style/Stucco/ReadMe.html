Dialog Skin - Style - Stucco

 

These are the Stucco style Dialog skins. They are based on the skin used in the original Lotro Grey Skin UI with a Stucco "texture" added to the background of the window.  The alternate versions of the skin were created by colorizing the basic skin style.
