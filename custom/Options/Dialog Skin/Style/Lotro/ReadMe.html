Dialog Skin - Style - Lotro

 

These are the Lotro style Dialog skins. They are based on ones originally developed for the Lord of the Rings Online game.

 

Lotro - Gloria Victis

This is the Lotro Gloria Victis style Dialog skin.  The skin is based on one used in the Gloria Victus Dragon UI for the Lotro game.

 

 

Lotro - Grey Skin

This is the Lotro Grey Skin style Dialog skin.  The skin is based on one used in the Grey Skin UI for the Lotro game.

 

 

Lotro - Wiggle

This is the Lotro Wiggle style Dialog skin.  The skin is based on the one used in Wiggle's Lotro UI for DAOC.

 

 

 

Note: Although it is not necessary, you should match the Button and Dialog skin styles for best appearance.
