Dialog Skin - Style - War

 

These are the War style Dialog skins. They are based on images found on Mythic's Warhammer Beta web site.

 

War - Dark

This is the War Dark style Dialog skins. The background of the skin has a 49% transparency level.

 

 

War - Solid

This is the War Solid style Dialog skins. The background of the skin is solid and non transparent.

 

 

 

Note: Although it is not necessary, you should match the Button and Dialog skin styles for best appearance.
