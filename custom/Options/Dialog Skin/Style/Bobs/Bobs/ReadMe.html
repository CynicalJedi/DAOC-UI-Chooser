Dialog Skin - Style - Bobs (Original)

 

This option allows you to choose this style for the Dialog Skin.

 

This is the original Bobs style Dialog skin. The background of the skin is based on the skin used in the Icz UI. The frame is based on the one in Mythic's original Atlantis UI.
