Dialog Skin - Style - Quillixx

 

These are the Quillixx style Dialog skins.  They are based on the ones in the Quillixx UI.

 

Quillixx - Dark

This is the Quillixx Dark style Dialog skins.  The background of the skin has a 52% transparency level.

 

 

Quillixx - Light

This is the Quillixx Light style Dialog skins.  The background of the skin has a 20% transparency level.

 

 

Quillixx - Solid

This is the Quillixx Solid style Dialog skins.  The background of the skin is solid and non transparent.

 

 

 

Note: Although it is not necessary, you should match the Button and Dialog skin styles for best appearance.
