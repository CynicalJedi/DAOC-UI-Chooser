Crafting (Future)

 

The Crafting window is a custom window that can be used as a guide during crafting.

 

The window will appear when you right click on the button labeled "Crafting" in the Main Menu or Command windows of the user interface.

 

- Conceal

There is nothing shown when the window is concealed.  The window is invisible with no graphics or controls to minimize the amount of memory used.  Nothing will appear on the screen when you attempt to activate the window.
