Chat

 

The Chat window is used to interact with the game client and display system and chat text messages.

 

The window is static and will always appear on your display unless it has been closed.

 

- Conceal

This option caused the game client to crash so it was disabled.

 

- Disable

This option allows you to enable the close button in the top right corner of the Chat windows.

 

 

Style 1

 

 

Style 2

When you hover over this style of chat window with your mouse, the background of the window will darken to make it easier to see the chat text.

 

 

Style 3

When you hover over this style of chat window with your mouse, the background of the window will NOT change.
