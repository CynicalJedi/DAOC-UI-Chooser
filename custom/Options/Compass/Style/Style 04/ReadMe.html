Compass - Style - Style 04

 

This option allows you to choose this style for the Compass window.

 

This is the combination small and large normal "strip" style compass.

 

 

 

Use the Control-C keyboard combination while in game to toggle between the small and large compass windows.

 

 

Note:  The normal frame is automatically disabled when this style is chosen because the game client does not support having both a small and large frame active at the same time.
