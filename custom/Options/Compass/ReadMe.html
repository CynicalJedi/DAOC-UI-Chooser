Compass

 

The Compass window is used to show our characters direction of travel as you move around in the game.

 

The window will appear when you click on the button labeled "Compass" in the Main Menu or Command windows of the user interface.  You can also use the Control-C keyboard combination (default) while in game to toggle the compass window it's off, small, or large states.

 

Normal

These options allow you to change the settings for the normal strip style compass.

 

Round

These options allow you to change the settings for the round style compass.

 

Style

There are several styles for the Compass window that you can choose.  These options allow you to change the window style.
