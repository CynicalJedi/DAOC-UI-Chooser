Performance

 

The Performance window is used to display a lag meter.

 

The window will appear when you click on the button labeled "Lag Meter" in the Main Menu or Command windows of the User Interface.  In addition you can use the shift P (default) keyboard combination to activate the window. Pressing the shift P combination will toggle the window from small to large to off.

 

The meter is arranged in normal order representing Performance (Packet Loss), Lag (Ping Time), and then Frame (Frames per Second).  The indicators change from green to yellow then red as the lag increases and performance/frame rate drops.

 

- Conceal

There is nothing shown when the window is concealed.  The window is invisible with no graphics or controls to minimize the amount of memory used.  Nothing will appear on the screen when you attempt to activate the window.
