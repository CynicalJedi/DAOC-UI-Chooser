Realm War

 

The Summary window is used to display information about the player health, endurance, and power status. Additional information such as player XP, concentration, master level, champion level, and realm level status may also be shown.

 

The window is static and will always appear on your display unless it has been disabled.

 

Key

The Realm War map Key window is used to show what the various symbols on the Realm War maps represent.

 

Maps

The Realm War maps are used to show the terrain for each realm war area.  There are several styles for the Realm War maps that you can choose.

 

Markers

The Realm War markers are used to show the location of your group members in the map window.  There are several styles for the Realm War markers that you can choose.

 

Relic Gate Links

The relic gate links are used to show which combination of keeps, when taken, will open a realm's relic gates.  There are several styles for the Realm War Relic Gate Links that you can choose.

 

Style

There are several styles for the Realm War window that you can choose.

 

Symbols

The realm symbols are used with the Realm War windows to designate which realm map is currently being displayed.
