Realm Bonuses

 

The Realm Bonuses window is used to display bonus information that is received by your entire realm for owning keeps, towers, and relics.

 

The window will appear when you click on the button labeled "Rlm Bonus" in the Main Menu or Command windows of the User Interface.

 

- Conceal

There is nothing shown when the window is concealed.  The window is invisible with no graphics or controls to minimize the amount of memory used.  Nothing will appear on the screen when you attempt to activate the window.
