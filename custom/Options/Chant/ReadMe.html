Chant

 

The Chant window is a custom window used to show the current "chant" type spell (Speed, Endurance, PBT, etc. ) that you are playing.

 

The window will appear when you right click on the button labeled "Chant" in the Main Menu or Command windows of the user interface.

 

Note:  The dual Chant window options were removed as the game client no longer supports them.

 

Background

There are two types of backgrounds for the Chant window that you can change.  These options allow you to choose the background style.

 

Style

There are several styles for the Chant window that you can choose.  These options allow you to choose the style for the Chant window.
