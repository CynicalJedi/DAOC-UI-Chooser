Chant - Style

 

There are several styles for the Chant window that you can choose.

 

Note:  The dual Chant window options were removed as the game client no longer supports them.

 

- Conceal

There is nothing shown when the window is concealed.  The window is invisible with no graphics or controls to minimize the amount of memory used.  Nothing will appear on the screen when you attempt to activate the window.

 

- Disable

Style 2

The style window contains only a Sprint button used to toggle sprint mode on/off.

 

 

Style 3

The style window contains a Sprint button used to toggle sprint mode on/off and a shows a single chant.

 

 

Style 4

The style window contains a Sprint button used to toggle sprint mode on/off and a shows a single chant.

 

 

Style 5

The style window contains only a Sprint and Attack button used to toggle sprint and attack modes on/off.

 

 

Style 6

The style window contains Sprint and Attack buttons used to toggle sprint and attack modes on/off and shows a single chant.

 

 

Style 7

The style window contains Sprint and Attack buttons used to toggle sprint and attack modes on/off and shows a single chant.
