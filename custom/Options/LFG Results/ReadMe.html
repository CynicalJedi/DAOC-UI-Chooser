LFG Results

 

The LFG Results window is used to display a list of players who are flagged as looking for a group.

 

The window will appear when you click on the button labeled "Show LFG" button at the top of the Status Group window. Click on a Player's name to select them. Click on the Send Message button to send a message to the selected player.

 

- Conceal

There is nothing shown when the window is concealed.  The window is invisible with no graphics or controls to minimize the amount of memory used.  Nothing will appear on the screen when you attempt to activate the window.
