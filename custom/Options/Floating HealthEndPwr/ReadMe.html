Floating HealthEndPwr

 

There are several styles for the Floating Health Endurance and Power windows that you can choose.

- Conceal

There is nothing shown when the windows are concealed.  The windows are invisible with no graphics or controls to minimize the amount of memory used.  Nothing will appear on the screen when you attempt to activate the windows.
