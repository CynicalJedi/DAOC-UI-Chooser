Quickbar

 

Quickbars are used to access various abilities, spells, and combat styles.  You can have up to three quickbars visible on your screen at a time.

 

The Quick Bar window is still an oldstyle bitmap window in the UI and there is not much we can do to customze it.  When you select a Dialog skin style, an oldstyle bitmap will be activated that matches the skin style you choose.  This option allows you to override the Quick Bar that defaults with each skin and replace it with a semi-transparent one.

 

Note:  You need to ReApply this setting (select it again) whenever you change the Dialog Skin style.

 

- Disable

This option allows you to Disable the transparent Quick Bar.  It will re-enable the original Quick Bar for the current Dialog Skin style.
