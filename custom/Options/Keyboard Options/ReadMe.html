Keyboard Options

 

The Keyboard Options window is used to display interactive Keyboard configuration information.

 

The window will appear when you click on the button labeled "Keyboard Options" in the Main Menu or Command windows of the user interface.  In addition you can type the command "/Keyboard" to activate the window.  The display shows the various Actions and the Assigned Key for the action.  To clear the assigned key for an action, select the action and click on the Clear Current Key button.  To change the assigned key for an action, click on the currently Assigned Key for the action and then press the key on your keyboard.  The Alt and Ctrl Key as Modifier buttons are used when assigning a new key to an action to change the selected new key.  The FPS, RPG, and MMO Default buttons are used to reset all the Assigned Keys to predefined settings.  The Accept button is used to accept the new keyboard settings and close the window.  The Cancel button is used to ignore any keyboard changes and close the window.

 

- Conceal

There is nothing shown when the window is concealed.  The window is invisible with no graphics or controls to minimize the amount of memory used.  Nothing will appear on the screen when you attempt to activate the window.
