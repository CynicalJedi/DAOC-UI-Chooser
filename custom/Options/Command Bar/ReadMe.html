Command Bar

 

The Command Bar is a custom window used to show a list of common command buttons.

 

The window will appear when you right click on the button labeled "Cmd Bar" in the Main Menu or Command windows of the user interface.

 

Background

There are two types of backgrounds for the Command Bar window that you can change.  These options allow you to choose the background style.

 

Style

There are several styles for the Command Bar window that you can choose.  These options allow you to choose the style for the Command Bar window.
